# observer 4375

地震情報を表示するwebアプリケーションです。

ほぼ趣味での開発になります。何かあればissueまでお願いします。

---

### 表示情報

- 地震感知情報

- 最新の地震情報

- 震度分布図

- 過去の地震情報

- 津波情報



### その他
